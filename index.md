### 開発関連

#### Vim

##### macOSでlua付きのVimをインストール
```bash
$ brew install lua
$ brew install vim --with-lua
$ vim --version | grep lua
+cursorshape       +lua               +ruby              -X11
```

##### Ubuntuでlua付きのVimをインストール
```bash
$ sudo apt-get install vim-gnome
```
